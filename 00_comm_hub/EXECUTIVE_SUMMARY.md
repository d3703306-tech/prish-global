## 🎯 Project Overview

---

## 📦 Deliverables

### ✅ Core Features
1. **Event Dashboard** - Browse cultural events with category filtering
2. **RSVP System** - Join events with secure email handling
3. **Memories Gallery** - Professional masonry photo gallery

### ✅ Technical Stack
- **Frontend:** React 18 + Tailwind CSS + Vite
- **Backend:** Python FastAPI
- **Database:** JSON file-based (events.json, rsvps.json, images.json)
